大文件

  • 打开大文件
    • glogg(只是查看)
    • JOE(可编辑)

打开大文件

glogg(只是查看)

安装

sudo apt-get install glogg

使用

glogg text.txt

JOE(可编辑)

安装

sudo apt-get install joe

使用

joe test.txt

使用joe打开文件后可以直接编辑文件, 其他操作例如跳转、搜索等需要配合 ctrl 键来使用。使用 ctrl + k + h来打开帮助
